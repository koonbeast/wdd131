*Overview*
Purpose
[This website is to provide an easily accessible database from anywhere for personal use. It will serve as a personal list made from user input. The input will only be saved for individual users who have access through their login credentials. Users can save titles and artists, and always be able to come back to see the things they've saved. There will be a shown list and a counter of how many rows of information has been saved. The user can also search or filter their entries in the list.]
Audience
[Music lovers and individuals who don't want to lose a song they really enjoyed or want to save to listen to another time.]

Dynamic elements
[Ability to add information to a main list and the list updates with the information. A counter of how many entries are on the list. The entry form, specifically the name input box, also acts as a search to filter and update the list to see if the name already exists in the list. Any other javascript will be secondary to the idea.]
